树形组件

用嵌套的数组表示出树形组件的数据，以树形组件的数据为参数新建一个树形组件，再将其渲染到某个元素上即可

实例：

选择的项：

未选择 

                //code
                

        var treeData = [
            {
                label: 'C:/',
                children: [
                    'Program Files',
                    {
                        label: 'Windows',
                        children: [
                            'System',
                            'System32',
                            'Fonts'
                        ]
                    }
                ]
            },
            {
                label: 'D:/',
                children: [
                    'wqswqdwq',
                    'sadasdasd',
                    'adascxzcxz'
                ]
            }
        ];

        var tree = new Tree(treeData);
        tree.render($('#demo-tree')[0]);
        tree.onchange = function () {
            $('.display-value span')[0].innerText = this.$value.innerText;
        };
